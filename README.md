Comment Component 

Tech Stack used : 
1) HTML
2) Javascript
3) TailwindCSS

Functionality developed: 
1) Dialog open on comment button click.
2) Dialog close on X button click.
3) Scrollable Comment dialog section.
4) Add a new comment(suggestion).
5) Delete a comment from the scrollable section.
6) Edit a comment from the scrollable section.
7) Show the size of the uploaded document.
